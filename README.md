# 🌐 Cisco Packet Tracer

Proyecto de prácticas y simulaciones de redes utilizando Cisco Packet Tracer desarrollado por **Isaac Esteban Haro Torres**.

---

## 📝 Descripción

Conjunto de prácticas y configuraciones de redes utilizando Cisco Packet Tracer, una herramienta de simulación de redes ampliamente utilizada en el ámbito educativo y profesional.

---

## ✨ Características

- Simulación de redes empresariales
- Configuración de routers y switches Cisco
- Protocolos de enrutamiento (RIP, OSPF, EIGRP)
- VLANs y trunking
- Configuración de servicios de red

---

## 🛠️ Stack Tecnológico

- Cisco Packet Tracer
- Protocolos de red (TCP/IP, UDP)
- Routing y Switching
- VLANs
